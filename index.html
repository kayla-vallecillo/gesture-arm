What it is

a pocket-sized
signal analyzer

this project puts a full-featured oscilloscope in the palm of your hand. built from off-the-shelf components and open firmware, it captures, processes, and streams real-time waveform data — then visualizes everything through this web interface.

hardware

built from
bare components

At its core lives a high-speed ADC paired with a microcontroller that handles buffering and compression. The analog front-end features adjustable gain and input protection, so the device can safely probe anything from millivolt sensor outputs to mains-level signals.

1M Samples / sec
12 Bit resolution
±50 Volt input range
8h Battery life
The pipeline

From probe tip
to your browser

Sampled data flows through a compression stage on-device, then transmits over USB or Wi-Fi to a local server. The server decodes, buffers, and serves the stream to this web frontend — keeping latency under 50 ms end-to-end.

live signal preview
The software

Web-native
visualization

The frontend is built around a Canvas-based renderer that draws waveforms at 60 fps. Trigger detection, time-base scaling, and measurement overlays all run client-side. No plugins, no installs — just open the page and probe.

Canvas API WebSockets Web Workers IndexedDB logging CSV export
Open source

Everything is
documented

Schematics, gerbers, firmware, and this web app are all public. The goal is a platform anyone can build, fork, and extend — whether you're debugging a sensor, teaching electronics, or just curious what your power supply actually looks like.
